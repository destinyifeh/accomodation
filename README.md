## About App

This is a next js web app created from scratch by me. This app is called accomodation.

Accomodation is a place where those in search for property to rent or buy get connected to the appropriate agents.

## Features

- Contact Form
- Registration Form
- Login Form
- Share Button
- Subscription Form
- And many more

## Stacks used

- Next js
- React
- Redux
- Node js
- MongoDB
- Tailwindcss
